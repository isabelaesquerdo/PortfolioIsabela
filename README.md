# Landing Page Profissional — Isabela Fernanda

Este é o repositório da minha Landing Page profissional (portfólio). O projeto foi construído do zero, utilizando práticas de HTML5 semântico e estilização moderna com CSS3 puro (sem frameworks como Bootstrap), focado em responsividade para computadores e celulares.

A página apresenta meu perfil multidisciplinar, integrando minhas competências na **Fisioterapia**, no **Técnico em Segurança do Trabalho** e na **Análise e Desenvolvimento de Sistemas (ADS)**.

---

## 🚀 Tecnologias Utilizadas

* **HTML5**: Estruturação semântica completa (`<header>`, `<nav>`, `<main>`, `<section>`, `<aside>`, `<footer>`).
* **CSS3**: Estilização moderna baseada no conceito de *Glassmorphism* (efeito de vidro fosco), paleta de cores personalizada, fontes do Google Fonts (*Outfit* e *Playfair Display*), ícones do *FontAwesome* e total responsividade (Mobile-First/Media Queries).

---

## 📦 Estrutura do Projeto

A página está organizada da seguinte forma:
* **Cabeçalho (`<header>`)**: Apresentação de perfil com meu nome completo, avatar personalizado e slogan.
* **Menu de Navegação (`<nav>`)**: Menu flutuante com links internos de scroll suave para fácil navegabilidade.
* **Sobre Mim (`#sobre`)**: Breve resumo sobre minha atuação profissional e acadêmica em Fisioterapia, Segurança do Trabalho e ADS.
* **Habilidades (`#habilidades`)**: Grid interativo destacando as 10 principais competências compartilhadas entre as minhas áreas de atuação.
* **Projetos (`#projetos`)**: Direcionamento e link rápido integrado ao meu GitHub para consulta de repositórios.
* **Informações Extras (`<aside>`)**: Curiosidades pessoais, hobbies e botões de atalho para minhas redes sociais (Instagram, LinkedIn e GitHub).
* **Rodapé (`<footer>`)**: Informações de contato e direitos autorais.

---

## 💻 Como Visualizar o Projeto Localmente

Você pode abrir a página diretamente no seu navegador.

### Método 1: Pelo Terminal (PowerShell)
Navegue até a pasta do projeto e execute:
```powershell
Start-Process "index.html"
```

### Método 2: Explorador de Arquivos
Basta dar dois cliques no arquivo `index.html` na pasta raiz do projeto.
